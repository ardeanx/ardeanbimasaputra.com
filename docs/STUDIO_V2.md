# Ardean CMS — Studio v2 (transformasi besar)

Acuan eksekusi. Keputusan user terkunci 2026-07-20 (ultracode, 6 arsitek).

## Keputusan terkunci

1. **DB**: Supabase → PostgreSQL lokal (`postgresql://postgres@127.0.0.1:5432/ardean`, trust). ✅ SELESAI.
2. **Studio = Admin Panel terpadu**, role-gated (member tipis / admin penuh). Bukan panel terpisah.
3. **Model konten**: `post.type` = VIDEO · AUDIO · POST · RESOURCE. Konten **gratis** untuk audiens.
4. **Products = tabel terpisah** (keputusan user, override arsitek). Product = entitas jual: title, slug, body, price, kind (DIGITAL|SOURCE_CODE|PHYSICAL|SERVICE), thumbnail, status, opsional postId. Store/order/entitlement → product-based. Content bukan dijual langsung.
5. **Resource** = file (`resource_file`) milik product (paid downloadable) atau post RESOURCE (showcase gratis).
6. **Editor** = chrome WordPress persis DI ATAS Tiptap (nol migrasi body). +ekstensi Tiptap. Buang tombol pajangan.
7. **Content page**: 5 tab (Semua/Video/Audio/Post/Resource), tabel fungsional TanStack Table+Virtual, visibility dropdown, bulk-select, hover-actions.
8. **Detail publik**: Video/Audio/Resource → `/watch?v=` (player); **POST → `/post/[slug]` immersive** (hero, TOC, author box, rating, prev/next, read-next).
9. **Settings** (7 kategori, semua fungsional): 1 baris JSON `app_setting` + cache Redis + `revalidateTag`. **Rahasia di `.env`**, DB simpan config publik+toggle+mode, UI tampilkan status.
10. **Shell**: hapus SEMUA garis pembatas (border-r/border-b/accent-bar), YouTube-Studio faithful.
11. **Stack**: @tanstack/react-table + react-virtual + react-query, react-hook-form + @hookform/resolvers (arktypeResolver, tetap ArkType), sonner, lucide-react, @dnd-kit, date-fns, recharts. +ekstensi Tiptap (drag-handle, table, text-align, youtube, color/highlight, html).

## Schema delta

- enum `post_type` → VIDEO|AUDIO|POST|RESOURCE (migrasi ARTICLE→POST via SQL recreate).
- enum `post_visibility` → PUBLIC|UNLISTED|PRIVATE (baru; terpisah dari `status`).
- `post` +kolom: slug(unique), mediaUrl, durationSec, repoUrl, excerpt, visibility.
- enum `product_kind` → DIGITAL|SOURCE_CODE|PHYSICAL|SERVICE. Tabel `product` baru.
- `resource_file` +productId (nullable), postId jadi nullable — file milik post ATAU product.
- `order` +productId, `entitlement` +productId — store product-based (postId dipertahankan utk kompat, path baru = product).
- tabel `rating` (postId,userId,stars) untuk POST.
- tabel `app_setting` (id=1, value jsonb) untuk Settings.
